# Dwight Meglan — Visual CV

Live at **https://dmeglan.github.io/visual-cv/**

A five-page visual curriculum vitae: experience map, an illustrated summary composed of
drawings from granted patents (each linked to its Google Patents record), a role &
contribution matrix, a hyperlinked patent portfolio, and grants/achievements.

- `index.html` — self-contained page (fonts and figures inlined); renders as fixed
  letter-size sheets and prints correctly to Letter landscape.
- `cv.pdf` — the same document as a PDF.

Built and maintained via Claude (Cowork); the regeneration pipeline lives in the
"dm cv" working folder. Contact: dmeglan@gmail.com · linkedin.com/in/dmeglan
